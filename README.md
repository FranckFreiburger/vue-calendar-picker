# vue-calendar
Calendar component

## Browser support
Same browser support as [Vue.js 2](https://github.com/vuejs/vue/blob/dev/README.md)

## Example - basic
```vue
<template>
	<calendar @action="action"></calendar>
</template>
```

## Install
```
npm install --save
```
